# BOLTT Website (CIS155 Web Development)

This is a website for BOLTT, a company idea for a "sovereign cloud" you run on
your own hardware, with an AI helper named Ebi. I built it for CIS155 Web
Development at Olympic College using plain HTML and CSS. 

## Pages

- `index.html` — Home page
- `how-it-works.html` — How BOLTT works, including a comparison table
- `about.html` — About the company and its mission
- `contact.html` — A "Request Access" page with a contact form

## Self-Assessment

I did not get peer feedback because im a bad student lol, so I looked over my own site and asked my boss whos an engineer: Here is where i landed. 

1. The site used the plain default font and had no icons, so it looked
   unfinished.
2. It was all paragraphs, with no easy way to compare BOLTT to a normal cloud,
   and no way for a visitor to contact us.

## Changes I Made

- **Font:** Added the Space Grotesk font from Google Fonts and used it across the
  whole site.
- **Icons:** Added a few FontAwesome icons, like an envelope and a paper plane on
  the contact page.
- **Table (extra credit):** Added a "BOLTT vs. Traditional Cloud" table on the
  How It Works page.
- **New page:** Added `contact.html` with a form (name, email, organization, and
  a message box), which brought the site to four pages.
- **Favicon (extra credit):** Used the BOLTT logo as the little icon that shows
  in the browser tab.
- **Other:** Made the logo link back to the home page and fixed some spelling.
