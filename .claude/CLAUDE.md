# CLAUDE.md — Boltt_website (CIS155 Web Dev)

## Project

Static landing page for Boltt sovereign cloud concept. Built as coursework for CIS155 Web Development at Olympic College.

- **Stack:** Plain HTML5 + CSS. No JS, no build tools, no framework.
- **Repo:** github.com/Boltthecat19/my-website (school work, not IP-sensitive — GitHub is fine here)
- **Branch:** main
- **Files:**
  - `index.html` — single-page layout (nav, hero, problem, solution, ebi sections)
  - `style.css` — global styles (dark bg `#1A1C1F`, aquamarine text)
  - `images/boltt-logo-dark2.svg` — logo asset
